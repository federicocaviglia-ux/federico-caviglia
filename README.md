# Portal Federico Caviglia · Dispositivo de orientación

Versión generada a partir de `federico_portal_multi.zip`.

## Enfoque
No es un CV tradicional ni una web de fantasía. Es un dispositivo de orientación personal, docente, tecnológico, creativo y profesional basado en el método científico.

Frases guía:
- Todo se resuelve en función de lo que sabemos.
- Mientras más sabemos, mejor decidimos.
- Ideas de hoy para soluciones de mañana.

## Estructura
- `index.html`: estación central del portal.
- `pages/quienes-somos.html`: red abierta de recorridos, saberes y colaboradores.
- `pages/metodo.html`: método científico aplicado como orientación.
- `pages/servicios.html`: servicios digitales, Drive, IA, QR, Apps Script, difusión y mantenimiento.
- `pages/proyectos.html`: proyectos terminados, activos y en espera de desarrollo/adaptación.
- `pages/recursos.html`: recursos impresos, evidencias de aula y reconocimientos.
- `pages/herramientas.html`: mapa de lenguajes, plataformas y territorios técnicos.
- `pages/referentes.html`: método científico, divulgación e imaginación científica.
- `pages/supermediaforge.html`: departamento creativo.
- `pages/contacto.html`: contacto y mapa INET.

También se mantienen alias/continuidad:
- `pages/metodologia.html`
- `pages/producciones.html`
- `pages/reconocimientos.html`
